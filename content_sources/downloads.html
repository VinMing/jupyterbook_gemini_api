下载或安装库以访问 Gemini

下载或安装库以访问 Gemini#

本页介绍了如何下载或安装用于访问 Gemini 的库。

Tip

注意 ：如果您刚开始接触 Gemini API，请尝试学习某个快速入门（例如Python 快速入门）。

Python

  • SDK：pip install google-generativeai

  • 低层级客户端库：pip install google-ai-generativelanguage

Go

  • 运行 go get github.com/google/generative-ai-go

Node.js

  • npm install @google/generative-ai

Web

  • 将import { GoogleGenerativeAI } from 'https://esm.run/@google/generative-ai'添加到您的 Web 应用

Swift

  • 使用 Swift Package Manager 将generative-ai-swift添加到您的 Xcode 项目。

Android

  • 将依赖项implementation("com.google.ai.client.generativeai:generativeai:0.1.1")添加到您的 Android 项目中。
